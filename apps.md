## Deployed Applications

### GhostNet — Ephemeral Peer-to-Peer WebRTC Chat
**Live:** https://ghostnet.app  
GhostNet is a real-time chat application built on **pure WebRTC mesh networking**. It demonstrates zero-friction access (no accounts), ephemeral “ghost” messages, and responsive real-time UX.

**Highlights**
- WebRTC P2P mesh architecture (no centralized chat server)
- Ephemeral messages + optional persistent session messages
- Built and deployed as a hackathon-grade, production-ready demo
